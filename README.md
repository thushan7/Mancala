# Mancala

This program simulates games of Mancala between two players.

## Getting Started

### Dependencies

There are no additional libraries required to run this program.

### Executing program

*1. Build the program
```
gradle build
```
*2. Run the program
```
java -cp build/classes/java/main ui.TextUI
```
*Expected Output
```
Enter Player 1 Name: 
```
